Наши программы

Для начинающих

Курсы английского с нуля — преодолеваем языковой барьер. Он подойдет тем, кто никогда не учил язык, не знает азов грамматики и не может составить на английском даже самое простое предложение.

Для взрослых

Хотите научиться говорить на английском языке? У нас вы можете пройти курсы английского языка для взрослых. Наши опытные преподаватели помогут вам освоить основы языка и совершенствовать свои навыки.

Для продвинутых

По окончании этого уровня вы сможете свободно и бегло говорить практически на любую тему. Получаемые навыки: свободное владение разговорным языком в разных ситуациях, большой словарный запас, полное знание грамматики.
